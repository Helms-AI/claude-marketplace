- Touch-friendly controls

## Updated Implementation Plan Priorities

Given mobile is HIGH priority, adjust phase ordering:

| Phase | Component | Mobile Impact |
|-------|-----------|---------------|
| 1 | Foundation (CSS Grid) | Critical - mobile-first breakpoints |
| 2 | Activity Bar (Right) | Critical - hamburger menu for mobile |
| 3 | Tree Views | Medium - works on mobile with adjustments |
| 4 | Tab System | Critical - horizontal scroll for mobile |
| 5 | Bottom Panel | Critical - bottom sheet for mobile |
| 6 | Conversation | Low - already responsive |
| 7 | Status Bar | Low - simplify for mobile |
| 8 | Command Palette | Medium - modal works on mobile |
| 9 | Keyboard Shortcuts | Low - touch alternative needed |
| 10 | Polish | High - touch gestures, swipe support |

## Responsive Breakpoints

```css
/* Mobile first */
:root {
  --sidebar-width: 0;           /* Hidden by default */
  --activitybar-width: 0;       /* Hidden, replaced by hamburger */
  --panel-height: 0;            /* Bottom sheet */
}

/* Tablet (768px+) */
@media (min-width: 768px) {
  :root {
    --sidebar-width: 200px;
    --activitybar-width: 40px;
    --panel-height: 150px;
  }
}

/* Desktop (1024px+) */
@media (min-width: 1024px) {
  :root {
    --sidebar-width: 280px;
    --activitybar-width: 48px;
    --panel-height: 200px;
  }
}

/* Large Desktop (1440px+) */
@media (min-width: 1440px) {
  :root {
    --sidebar-width: 320px;
    --panel-height: 250px;
  }
}
```

## Touch Gestures (Mobile)

| Gesture | Action |
|---------|--------|
| Swipe right from left edge | Open sidebar |
| Swipe left on sidebar | Close sidebar |
| Swipe up from bottom | Expand bottom panel |
| Swipe down on panel header | Collapse panel |
| Long press on message | Context menu |
| Pinch on graph | Zoom in/out |
| Two-finger pan on graph | Pan view |

**Final Layout Order (Left to Right)**:

**Bottom Panel**: Visible by default (200px)

**Tabs**: Unlimited count, horizontal scroll when overflow
