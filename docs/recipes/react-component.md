# Recipe: React Component

Create a React component with design specs, implementation, and tests.

## Overview

| Property | Value |
|----------|-------|
| **Time** | ~5 minutes |
| **Difficulty** | ⭐ Beginner |
| **Domains** | user-experience → frontend → testing |

## Command

```bash
/pm-recipe react-component --name="UserCard"
```

## What It Does

1. **User Experience** (1min)
   - Component specification
   - Visual style guide

2. **Frontend** (3min)
   - React component
   - TypeScript types
   - Storybook story

3. **Testing** (1min)
   - Unit tests

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `--name` | Yes | Component name |
| `--variant` | No | "atom", "molecule", "organism" |
