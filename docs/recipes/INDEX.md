# 📖 Workflow Recipe Cookbook

Pre-built workflow patterns for common development tasks.

## Quick Start

```bash
# List all recipes
/pm-recipe list

# Run a recipe
/pm-recipe full-stack-feature --name="User Profile"

# View recipe details
/pm-recipe show api-endpoint
```
