# Recipe: i18n Setup

Add internationalization to your frontend application.

## Overview

| Property | Value |
|----------|-------|
| **Time** | ~8 minutes |
| **Difficulty** | ⭐⭐ Intermediate |
| **Domains** | frontend |

## Command

```bash
/pm-recipe i18n-setup --framework="react"
```

## What It Does

1. Install i18next and dependencies
2. Create i18n configuration
3. Set up locale files structure
4. Add language switcher component
5. Extract existing strings

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `--framework` | No | "react", "vue", "svelte" |
| `--languages` | No | Comma-separated language codes |
